Signature moment

Suspend — say the blast radius out loud

Ban is user-level, so it reaches every app in the suite (Decision 4). The confirm dialog enumerates the user's other memberships from a native read, so the admin sees exactly what the suspension touches before confirming — never presented as app-scoped.

!

Suspend Jane Mokoena?

Suspending bans this login and revokes every session immediately. Because a ban is set on the person — not this app — it blocks Jane from all apps she can reach:
M Module Demo App Admin, Editor
C CRM Editor
F Field Ops Viewer
A Analytics Viewer
Reinstating later restores access everywhere. To remove Jane from this app only, use Remove from app instead.
