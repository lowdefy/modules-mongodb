# Files — Vars

- **`bucket`** (required) — S3 bucket name for private file storage.
- **`collection`** — MongoDB collection name for file metadata. Default: `files`.
- **`public_bucket`** — S3 bucket name for public file storage.
- **`s3_region`** — AWS region for S3 buckets. Default: `af-south-1`.
- **`components`** — Overrides: `file_list`.
